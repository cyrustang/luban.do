Favicon Generator Guide

Note: This guide will help you create the necessary favicon files for your website to display properly on desktop browsers.

Option 1: Use an Online Favicon Generator (Recommended)

  1. Visit RealFaviconGenerator.net or Favicon-Generator.org
  2. Upload your existing icon-512x512.png file
  3. Download the generated favicon package
  4. Extract and copy the following files to your public directory:
    • favicon.ico
    • favicon-16x16.png
    • favicon-32x32.png

Option 2: Convert Icons Manually

  1. Open icon-512x512.png in an image editor (e.g., Photoshop, GIMP, Paint.NET)
  2. Resize to 16x16 pixels and save as favicon-16x16.png
  3. Resize to 32x32 pixels and save as favicon-32x32.png
  4. For favicon.ico, you'll need to convert one of these PNG files to ICO format using a converter like Convertio or Online-Convert

Option 3: Use the command line (for technical users)

If you have ImageMagick installed on your system, you can run these commands:

  # Generate 16x16 PNG
  magick convert public/icon-512x512.png -resize 16x16 public/favicon-16x16.png
  
  # Generate 32x32 PNG
  magick convert public/icon-512x512.png -resize 32x32 public/favicon-32x32.png
  
  # Generate ICO file with multiple sizes
  magick convert public/icon-512x512.png -define icon:auto-resize=16,32,48,64 public/favicon.ico

Verification

Once you've created these files, make sure they're placed in your project's public directory:

  • public/favicon.ico
  • public/favicon-16x16.png
  • public/favicon-32x32.png

Your website's layout.tsx already has the necessary link tags to reference these files.
